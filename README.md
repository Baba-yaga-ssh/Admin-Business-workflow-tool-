AdminFlow

AdminFlow is a Python CLI tool I built to automate repetitive file and data management tasks.

Current Features

- CSV cleaning
  - Removes duplicate rows
  - Removes empty rows
  - Standardizes inconsistent values
  - Generates a summary report

- File sorting
  - Organizes files by extension into separate folders

- Bulk file renaming
  - Renames multiple files using a custom naming pattern

Status

This is currently an internal tool that I use for automation and service work. A public source code release is not available at this time.
